
# Broker , Agent , Proxy , Delegator

> Create Time : 2017年4月30日 Ref : http://blog.csdn.net/cyrus9669/article/details/52679794

* Broker - 经纪人，为促成他人交易，充当订阅中间人，为委托方提供订阅的信息、机会和条件的主体。Broker是一个独立主体，但没有自主决策能力，只负责订阅过程。
* Agent - 代理人，是一个独立个体，负责完成任务，但不负责执行任务，Agent具有一定的自主决策能力，如对服务请求的选择。
* Proxy - 代理，是指行为代理，不是一个主体。Proxy是完全的传递者，如请求和响应的转发，操作控制的传递。
* Delegator - 委托，接受请求的对象将请求委托给另一个对象来处理。
